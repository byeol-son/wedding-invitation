# 갤러리 이미지 폴더

이 폴더에 갤러리 사진들을 업로드해주세요.

## 사용 방법

1. 이 폴더에 사진 파일들을 업로드하세요 (예: `01.jpg`, `02.jpg`, `03.jpg` ...)
2. `script.js` 파일을 열어서 `INVITE.gallery.images` 배열에 파일명을 추가하세요:
   ```javascript
   gallery: {
     images: ["01.jpg","02.jpg","03.jpg","04.jpg"]
   }
   ```

## 파일명 규칙

- 숫자 순서대로 명명하는 것을 권장합니다: `01.jpg`, `02.jpg`, `03.jpg` ...
- 파일명은 `script.js`의 배열에 있는 이름과 정확히 일치해야 합니다
- 대소문자를 구분하므로 주의해주세요

# 이미지 업로드 가이드

이 폴더에 다음 이미지들을 업로드해주세요.

## 📤 이미지 업로드 방법

### 방법 1: Finder 사용 (가장 쉬운 방법)
1. Finder를 열고 이 프로젝트 폴더(`wedding-invitation`)로 이동
2. `images` 폴더를 열기
3. 업로드할 이미지 파일을 Finder에서 `images` 폴더로 **드래그 앤 드롭**
   - 메인 사진 → `images/main.jpg`로 이름 변경 후 드롭
   - 갤러리 사진 → `images/gallery/` 폴더로 드롭 (파일명: `01.jpg`, `02.jpg` 등)

### 방법 2: Cursor IDE 사용
1. Cursor 왼쪽 파일 탐색기에서 `images` 폴더를 찾기
2. 업로드할 이미지 파일을 Finder에서 찾기
3. 이미지를 Cursor의 `images` 폴더로 **드래그 앤 드롭**
4. 파일명이 올바른지 확인 (예: `main.jpg`, `01.jpg` 등)

### 방법 3: 터미널 사용
```bash
# 메인 이미지 복사
cp /경로/대상/이미지.jpg images/main.jpg

# 갤러리 이미지 복사
cp /경로/대상/사진1.jpg images/gallery/01.jpg
cp /경로/대상/사진2.jpg images/gallery/02.jpg
```

### 방법 4: 파일 탐색기에서 직접 복사
1. 사진이 있는 폴더를 열기
2. 이미지 파일을 선택하고 `Cmd + C` (복사)
3. Finder에서 `wedding-invitation/images` 폴더로 이동
4. `Cmd + V` (붙여넣기)
5. 파일명을 올바르게 변경 (예: `main.jpg`)

## 필요한 이미지 파일

### 1. 메인 사진 (필수)
- **파일명**: `main.jpg`
- **위치**: `images/main.jpg`
- **용도**: 청첩장 상단 히어로 섹션의 배경 이미지
- **권장 크기**: 가로 1200px 이상, 세로 1600px 이상 (세로형 권장)
- **설명**: 이 이미지가 청첩장의 첫 화면에 표시됩니다.

### 2. 갤러리 사진들
- **위치**: `images/gallery/` 폴더 안
- **파일명**: `01.jpg`, `02.jpg`, `03.jpg`, ... (숫자 순서대로)
- **용도**: 갤러리 섹션에 표시되는 사진들
- **설정**: `script.js` 파일의 `INVITE.gallery.images` 배열에 파일명을 추가하세요.
  ```javascript
  gallery: {
    images: ["01.jpg","02.jpg","03.jpg",...]
  }
  ```
- **권장 크기**: 정사각형 또는 1:1 비율 권장 (최소 800x800px)

### 3. 소셜 미리보기 이미지 (선택)
- **파일명**: `og.jpg`
- **위치**: `images/og.jpg`
- **용도**: 카카오톡, 페이스북 등에서 링크 공유 시 표시되는 미리보기 이미지
- **권장 크기**: 1200x630px (가로형, 1.91:1 비율)

## 파일 구조 예시

```
images/
├── main.jpg          (메인 히어로 이미지)
├── og.jpg            (소셜 미리보기 이미지, 선택)
└── gallery/
    ├── 01.jpg
    ├── 02.jpg
    ├── 03.jpg
    └── ...
```

## 주의사항

1. 파일명은 정확히 일치해야 합니다 (대소문자 구분)
2. 이미지 형식은 JPG, PNG 모두 가능하지만, 파일 확장자는 `.jpg` 또는 `.png`로 통일해주세요
3. 파일 크기가 너무 크면 로딩이 느려질 수 있으니, 웹용으로 최적화된 이미지를 사용하세요
4. `script.js`의 `gallery.images` 배열에 추가한 파일명과 실제 파일명이 일치해야 합니다
